## 🚀 Getting Started

### Quick Setup (3 steps)
```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Download language model
python -m spacy download en_core_web_sm

# 3. Run the app
streamlit run app.py
```

### Windows Users
Simply double-click `run.bat` to start!
